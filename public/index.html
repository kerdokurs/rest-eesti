Eesti REST API

Tere tulemast kasutama Eesti REST API-d.

Eesti REST API on API, kust saab infot Eesti kohta.

Päringud

Kõik


        GET eesti.kerdo.me/v1/all
Parameetrid

        fields=[id,nimi,pindala,rahvaarv] - komaga eraldatud väljad

Maakonnad


        GET eesti.kerdo.me/v1/maakonnad
Parameetrid

        fields=[id,nimi,pindala,rahvaarv,keskus] - komaga eraldatud väljad

Linnad


        GET eesti.kerdo.me/v1/linnad
Parameetrid

        fields=[id,nimi,pindala,rahvaarv] - komaga eraldatud väljad

Üldised parameetrid


        limit={arv} - tulemuste arv (vaikesäte = 20)
        order={väli},{asc | desc} - sorteerimine välja järgi

Näited

Päring


        GET eesti.kerdo.me/v1/maakonnad?fields=nimi,rahvaarv&limit=5&order=rahvaarv,desc
